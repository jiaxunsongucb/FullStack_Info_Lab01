1.   Change Directory: cd
2.   Create a Directory: mkdir
3.   Rename a Directory: mv old_name new_name
4.   List the contents of a directory: ls
5.   Move a file to a new location: mv file_name /directory_name
6.   Copy a file to a new location: cp file_name /directory_name
7.   Search for a string in a file: grep string file_name
8.   Access the shell of a remote machine: ssh
9.   Remove a directory folder: rm -rf directory_name
